Semitono es un programa libre de código abierto. El código fuente está dispuesto en GitHub bajo el GPLv3.

Si tienes algún comentario o algunas sugerencias para mejorar Semitono, ¡me alegraría escucharlos! Puedes enviarme un correo electrónico a: andy@tck.mn.

Este programa utiliza:

  • bibliotecas desde el proyecto FFmpeg, bajo el GPLv2 o posterior
  • iconos de Material Design y la biblioteca Oboe, ambos licenciados por Google bajo el Apache 2.0
  • audio proporcionado por el proyecto University of Iowa Musical Instrument Samples
